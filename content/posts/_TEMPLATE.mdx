## Use H2 headings for main sections

Every `##` and `###` heading automatically appears in the article's Table of Contents,
so structure your article with them.

## Callout boxes

Use these to highlight important information:

<Note>This is a general note for extra context.</Note>

<Tip title="Pro tip">A helpful suggestion the reader can act on.</Tip>

<Warning title="Be careful">Something the reader should watch out for.</Warning>

<Info>Neutral, informational aside.</Info>

<Danger title="Critical">A serious warning about a dangerous action.</Danger>

## Code blocks

Add a language after the opening backticks for syntax highlighting. You can also add a
`title` and highlight specific lines with `{2-3}`:

```typescript title="example.ts" {2}
function greet(name: string) {
  return `Hello, ${name}!`; // this line is highlighted
}
```

## Images

Upload the image to `public/uploads/` first, then reference it. Paths start with
`/uploads/`. Basic image:

![A short description of the image](/uploads/example.png)

Or with a caption (recommended):

<Figure
  src="/uploads/example.png"
  alt="A clear description of what the screenshot shows"
  caption="Figure 1: An example screenshot"
/>

An image gallery:

<Gallery
  images={[
    { src: '/uploads/shot-1.png', alt: 'First screenshot', caption: 'Step one' },
    { src: '/uploads/shot-2.png', alt: 'Second screenshot', caption: 'Step two' },
  ]}
/>
